# Alke Wallet - Repositorio de Proyecto

Este repositorio contiene el código fuente de la aplicación **Alke Wallet**, una billetera digital interactiva desarrollada con tecnologías web estándar.

## Enlaces de Entrega

* **Prototipo Funcional (GitHub Pages):** [https://marchantfernanda5-wq.github.io/Alke-Wallet/](https://marchantfernanda5-wq.github.io/Alke-Wallet/)

* **Código Fuente:** [https://github.com/marchantfernanda5-wq/Alke-Wallet](https://github.com/marchantfernanda5-wq/Alke-Wallet)

## Descripción Técnica

El proyecto ha sido estructurado para cumplir con los requisitos de navegación y gestión de datos solicitados:

* **Interfaz:** Desarrollada con HTML5, CSS3 y el framework Bootstrap para asegurar un diseño responsivo.
* **Lógica:** Implementada con JavaScript (jQuery) para la gestión de eventos.
* **Persistencia:** Uso de LocalStorage para mantener el saldo, los movimientos y la agenda de contactos de forma local en el navegador.

## Instrucciones de Uso

1.  **Acceso:** En la pantalla de login, ingrese un correo electrónico y una contraseña (mínimo 4 caracteres).
2.  **Operaciones:** Desde el menú principal puede acceder a los módulos de depósito, envío de dinero e historial.
3.  **Simulación:** Todas las transacciones se reflejan en tiempo real en el saldo principal.
